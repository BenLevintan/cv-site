About This Game

Use your paddle to bounce the ball and destroy the colorful bricks. Clear the board to win. Miss the ball, and you restart. Simple rules. Unforgiving consequences.
FEATURES
Classic paddle movement with mouse controls. Colorful rows of bricks. Score tracking. Angle manipulation based on paddle hit location. Zero loading screens. Zero microtransactions. Zero loot boxes.
image placeholder
BUILT FROM SCRATCH
No game engine. No frameworks. Pure vanilla JavaScript and an HTML5 canvas element. The entire game logic fits in one file with a custom collision engine.
