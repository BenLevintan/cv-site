About This Game

Clear every brick with a single ball and a shrinking paddle. The arcade classic, no quarters needed.
CLEAR EVERY BRICK
Classic paddle movement with mouse controls. Colorful rows of bricks. Score tracking. Angle manipulation based on paddle hit location. Zero loading screens. Zero microtransactions. Zero loot boxes.
BUILT FROM SCRATCH
No game engine. No frameworks. Pure vanilla JavaScript and an HTML5 canvas element. The entire game fits in one file.
