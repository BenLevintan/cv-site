Matrix Mask Picker

Visual binary matrix editor for optical hardware calibration

View on GitHub

Context

In optical systems like adaptive optics or wavefront sensors, hardware components (such as actuators and lenslets) must be selectively enabled or disabled via binary mask configurations.

Editing these complex masks as raw JSON or CSV is incredibly error-prone. This tool gives hardware engineers a fast, visual, and clickable interface to actively manage those configurations.
