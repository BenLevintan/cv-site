Matrix Mask Picker

Visual binary matrix editor for optical hardware calibration

View on GitHub

Context

In optical systems like adaptive optics or wavefront sensors, hardware components (such as actuators and lenslets) must be selectively enabled or disabled via binary mask configurations.

Editing these complex masks as raw JSON or CSV is incredibly error-prone. This tool gives hardware engineers a fast, visual, and clickable interface to actively manage those configurations.

Visual Cell Toggling

JSON + CSV Support

Easily load structured JSON configurations and export flat CSV matrix maps.

Master/Slave Sync

Dual-panel view allowing engineers to overwrite a slave matrix via the master panel.

All On / All Off

Global reset functions for rapid blanket testing of hardware systems.

Save to CSV

Outputs standardized CSV strings optimized for optical hardware ingestion.

Two Matrix Formats

Adaptive grid system specifically designed to handle 12x12 and 15x16 arrays.
